# Demos

Minimal UI skins for the Puhemies workflow. Both demos use the same runtime functions and artifact folders.

## Install demo dependencies

```bash
python -m pip install -r demos/requirements-demo.txt
```

## TUI demo

```bash
python demos/tui_app.py --input data/samples/sample_messy.xlsx --interactive
```

## Streamlit demo

```bash
streamlit run demos/streamlit_app.py
```

## Streamlit Mapping Studio (V2)

```bash
streamlit run demos/streamlit_mapping_studio.py
```
